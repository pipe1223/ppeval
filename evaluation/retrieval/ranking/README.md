# Ranking evaluation

This directory contains ranking metrics used by retrieval evaluation.

Current implementation:
- `metrics.py`

Current metrics include:
- Precision@k
- Recall@k
- HitRate@k
- MRR
- Average Precision
- nDCG@k

These are useful for:
- document retrieval
- chunk retrieval
- citation retrieval
- GraphRAG context retrieval
