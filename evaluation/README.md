# Evaluation framework

This directory contains the PP Evaluation framework.

Subareas:
- `visual/` — visual and VLM evaluation
- `retrieval/` — ranking, RAG, and GraphRAG evaluation
- `shared/` — reusable metrics, schemas, and helpers
- `online/` — pairwise, judge-based, validator, and human review helpers

Backwards compatibility is preserved for the original visual evaluator through top-level compatibility wrappers.
